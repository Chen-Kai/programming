JSON官网给出的[Java](http://www.json.org/java/index.html)实现，主要的类有：
- [JSONObject](http://www.json.org/javadoc/org/json/JSONObject.html)
与JSON格式中的对象相对应，是一些键值对的集合。

- [JSONArray](http://www.json.org/javadoc/org/json/JSONArray.html)
与数组相对应，是一组值序列。

- [JSONStringer](http://www.json.org/javadoc/org/json/JSONStringer.html)
与字符串相对应，用于生成字符串。

- [JSONWriter](http://www.json.org/javadoc/org/json/JSONWriter.html)
负责生成JSON文件（符合JSON格式）。

- [JSONTokener](http://www.json.org/javadoc/org/json/JSONTokener.html)
解析JSON文件

- [JSONException](http://www.json.org/javadoc/org/json/JSONException.html)
当JSON格式有误时抛出异常

- [JSONString](http://www.json.org/javadoc/org/json/JSONString.html)
序列化接口
